# Xboost
https://xboostproductivity.vercel.app/

> Save smarter. Organize faster. Discover more with Xboost.

Xboost is a modern bookmark manager and content curation platform that helps users save, organize, and manage links, videos, and online resources from a single dashboard.

## Features

- 🔖 Save and manage bookmarks
- 🎥 Store and organize video links
- 📁 Create custom collections and categories
- 🔍 Powerful search and filtering
- 👤 Secure user authentication
- ☁️ Cloud-based data storage
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and intuitive user experience
- 🌙 Modern and clean interface

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase (Authentication & Database)
- Progressive Web App (PWA)

## Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm
- Supabase account

### Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/xboost.git
```

2. Navigate to the project folder

```bash
cd xboost
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```
